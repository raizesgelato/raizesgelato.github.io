# Raízes Gelato — site estático

Estrutura pronta para subir no GitHub Pages.

## Arquivos principais
- `index.html`
- `styles.css`
- `assets/logo-raizes.jpeg`
- `assets/pattern-raizes.jpeg`

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. No GitHub, vá em **Settings > Pages**.
4. Em **Build and deployment**, selecione:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / `/root`
5. Salve.

## Domínio personalizado
Depois de o site estar publicado:
1. Em **Settings > Pages**, adicione `raizesgelato.com.br` em **Custom domain**.
2. No Registro.br, aponte o domínio para o GitHub Pages.
